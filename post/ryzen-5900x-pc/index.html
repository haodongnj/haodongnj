台式机折腾记

2021-06-21

配置

趁着618攒了一台主机，犹豫了很久还是决定上车AMD，配置如下：

  • CPU: AMD Ryzen-9 5900x
  • 主板：TUF Gaming X570-Plus WIFI
  • 硬盘：
    • 1TB SSD KC2500
    • 4TB HDD 西部数据蓝盘
  • Fury 32GB DDR4 3600MHz
  • 美商海盗船AIR540中塔机箱
  • 美商海盗船HX1200，1200W电源
  • GTX 750Ti显卡（等矿难）

装机的过程略去不表。记录一些配置工作：

  1. 安装了Windows 10和Ubuntu 20.04双系统，但是Ubuntu下可识别蓝牙但是无法识别WIFI，需要关闭Windows的快速启动功能，这样Ubuntu就可以愉快地连接WIFI了

  2. 为了方便使用vim，Ubuntu下把CapsLock交换为Ctrl：采用xcape即可

sudo apt-get install xcape

# make CapsLock behave like Ctrl:
setxkbmap -option ctrl:nocaps

# make short-pressed Ctrl behave like Escape:
xcape -e 'Control_L=Escape'

后发现xcape可能会出现失效的情况，比较麻烦，所以改用了桌面自带的工具：

  • Windows系统可采用PowerToys修改键盘映射，无需手动修改注册表
  • Ubuntu可用桌面自带的设置来修改键盘映射, gnome的话可以考虑gnome-tweaks工具
  1. 配置grub记住上次的启动项目
sudo vim /etc/default/grub

# put the following lines
GRUB_DEFAULT=saved
GRUB_SAVEDEFAULT=true

# then run
sudo update-grub
  1. 双系统时间差问题

使用下来AMD还是很香的，就是显卡不太给力，不过基本的功能算是有了。试着以12线程编译一些项目，速度还是很快的。

AMD, Yes!
